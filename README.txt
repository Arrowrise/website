ARROWRISE CO-WORKING SPACE — Website Package
=============================================

Files included:
  index.html              → Main website (open this in any browser)
  assets/
    images/               → Your space photos (space-01 to space-08)
                            Replace the Unsplash placeholder URLs in index.html
                            with these image paths (e.g. assets/images/space-01.png)
    docs/
      arrowrise_poster.pdf    → Poster
      arrowrise_brochure.pdf  → Brochure

HOW TO SWAP IN YOUR OWN PHOTOS
--------------------------------
In index.html, search for any line with:
  src="https://images.unsplash.com/..."
Replace the URL with the relative path to your image, e.g.:
  src="assets/images/space-03.png"
